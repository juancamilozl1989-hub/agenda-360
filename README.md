# Agenda 360

Sistema inteligente para la gestión de citas, clientes y automatización de procesos mediante inteligencia artificial.

> Proyecto desarrollado como parte de mi proceso de formación en Desarrollo de Software, aplicando buenas prácticas de arquitectura, desarrollo de APIs, bases de datos, control de versiones e integración con servicios externos.

## Estado del proyecto

🚧 En desarrollo

Actualmente el proyecto se encuentra en fase de construcción del MVP (Producto Mínimo Viable).

---

## Objetivo

Desarrollar una plataforma moderna, escalable y modular que permita gestionar citas, clientes, empleados y servicios, incorporando automatización e inteligencia artificial para optimizar los procesos de atención.

---

## Tecnologías utilizadas

- Python
- FastAPI
- PostgreSQL
- Git y GitHub
- Uvicorn
- Pydantic

> Durante el desarrollo se incorporarán nuevas tecnologías como Docker, autenticación JWT, integración con WhatsApp y servicios de inteligencia artificial.

---
